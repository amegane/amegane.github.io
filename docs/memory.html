    ページをカテゴリごとに分類したり、お気に入りに登録したりすることで、あなた好みのアプリへと進化していきます。

    詳細画面では、ページの内容を見ることはもちろん、画面をスワイプすることで別のページに切り替えることもでき、快適に操作が行えます。

    ロック画面や別のアプリを開きながらページを確認したいときは、ピン留め機能を使うことで、ページを通知欄で確認できます。
