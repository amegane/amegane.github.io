Memory プライバシーポリシー

このページでは雨鐘が提供するアプリ「Memory」(以下、本アプリケーション)のプライバシーポリシーを記載しています。

本アプリケーションで収集する情報

本アプリケーションでは、広告配信のためにGoogle AdMob、利用情報解析のためにGoogle Firebase Analyticsを使用しています。これらは個人を特定するために使用していません。

Google社のプライバシーポリシーは以下のリンクを参照してください。

Google プライバシーポリシー

バージョン2.0.0以降では、Google Drive APIを用いて、Google Driveにバックアップ用のファイルの保存、バックアップ用のファイルからデータを復元するために必要なGoogleアカウント情報の取得、保存を行います。 本アプリケーションは、Google API Serviceのユーザーデータポリシーに準拠し、Google Drive APIから受け取った情報を使用します。

Google API Servicesのユーザーデータポリシーは以下のリンクを参照してください。

Google API Services User Data Policy

本アプリケーションで取得された利用者情報は、アプリケーション内のみで使用され、第三者に提供することはありません。ただし、ユーザーの同意がある場合や法令に基づき開示が必要な場合はこの限りではありません。

プライバシーポリシーの変更について

本プライバシーポリシーの内容に関する変更は、ユーザーに通知することなく行えるものとします。本プライバシーポリシーは常に最新版が適用され、このページに掲載された段階で効力を生じるものとします。
